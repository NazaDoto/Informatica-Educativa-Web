Problemas Computacionales

Problemas Computacionales
Definición

Son cuestiones o proposiciones dudosas, que se tratan de aclarar o resolver.

Problemas - Tipos [I]

Problemas que buscan respuestas:
Se debe encontrar algo que se desconoce.

Problemas - Tipos [II]

Problemas que buscan pruebas:
Ya se sabe lo que se desea, solo se busca determinar la relación entre los datos y la respuesta.

Problemas - Tipos [III]

Problemas de computación:
Son problemas que buscan métodos mediante los cuales se pueda derivar una respuesta.

¿Cómo resolver un problema?

Necesitamos crear un algoritmo que al ejecutarlo nos devuelva la respuesta deseada.

Algoritmos

Algoritmos - Definición

Son una secuencia ordenada de pasos elementales, que lleva a la solución de un problema dado en un tiempo finito.

Algoritmos - Propiedades [I]

Tiempo de respuesta:
La ejecución de un algoritmo debe finalizar en una cantidad finita de pasos.

Algoritmos - Propiedades [II]

Generalidad:
Debe poder ser aplicado a una diversidad de casos del problema que contempla.

Algoritmos - Propiedades [III]

Ausencia de ambigüedad:
La representación de cada paso de un algoritmo debe tener una única interpretación.

Un algoritmo se puede expresar en:

-Lenguaje común
-Pseudocódigo
-Diagrama de flujo
-Lenguaje de programación

Lenguaje común

Es el lenguaje normal que hablamos y escribimos, es útil para comunicar un algoritmo a otra persona.

Pseudocódigo

Describe un algoritmo utilizando expresiones en lenguaje natural, pero aplicando la lógica de un programa, y de este modo, expresando su flujo de control.

Lenguaje de programación

Es la forma obligada de expresión de un algoritmo para que pueda ser leído, ejecutado y almacenado en la computadora.

Diagrama de flujos

Es un lenguaje gráfico, es útil para visualizar en forma rápida la secuencia lógica de pasos a seguir para un algoritmo.
